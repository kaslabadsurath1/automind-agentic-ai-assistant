# AutoMind - Agentic AI Assistant

AutoMind is an Agentic AI Assistant built using LangChain, Streamlit, and Python.  
The application demonstrates autonomous AI workflows with tool integration, conversational memory, document interaction, and streaming responses.

## Features

- Conversational AI assistant
- Chat with PDF/documents
- SQL database interaction
- Pandas DataFrame querying
- Streaming AI responses
- Agent memory handling
- Tool calling workflows
- Streamlit-based UI

## Tech Stack

- Python
- LangChain
- Streamlit
- OpenAI API
- SQLite
- Pandas
